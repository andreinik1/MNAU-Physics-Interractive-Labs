## 🚀 Про проєкт

**MNAU-Physics-Interractive-Labs** — це сучасний інтерактивний вебдодаток, створений для діджиталізації, візуалізації та автоматичної перевірки результатів лабораторних робіт з фізики для студентів МНАУ. Цей продукт повністю замінює застарілі паперові журнали та Excel-таблиці, автоматизуючи контроль за виконанням практичних завдань.

Студенти можуть взаємодіяти з віртуальними фізичними установками в реальному часі, спостерігати за симуляцією процесів та вносити дані у динамічні таблиці, які миттєво валідують правильність розрахунків.

### 🔬 Доступні лабораторні роботи:
1. **Визначення адіабатичного показника газу (Метод Клемана-Дезорма)** — інтерактивний симулятор манометра з компресором та сифоном. Візуальна частина базується на реальному приладі: у стані спокою ($h = 0$) рівень води зафіксовано на позначці $75 \text{ мм}$, а при нагнітанні тиску меніски рідини динамічно розходяться вгору та вниз від цієї стартової позиції.
2. **Дослідження теплового розширення твердих тіл (ТТР)** — лабораторна робота з автоматичним розрахунком коефіцієнта лінійного розширення $\alpha$. Включає гнучке налаштування кількості замірів (до 15 рядків) із повним збереженням уже введених студентом даних при зміні розміру таблиці.

## 📦 Встановлення та запуск локально

Щоб розгорнути та протестувати проєкт на своїй локальній машині, виконайте наступні кроки:

```bash
git clone [https://github.com/andreinik1/MNAU-Physics-Interractive-Labs.git](https://github.com/andreinik1/MNAU-Physics-Interractive-Labs.git)
cd MNAU-Physics-Interractive-Labs

Встановлення залежностей
npm install
# або
yarn install
Запуск локального сервера розробки
npm run dev
# або
yarn dev

Після успішного запуску відкрийте у браузері:

http://localhost:5173 
```

📐 Структура та архітектура лабораторних робіт

Кожен модуль лабораторної роботи стандартизований відповідно до архітектури застосунку та складається з трьох основних частин.

📝 Компоненти введення (LabTable.tsx)

Інтерактивна таблиця результатів вимірювань, яка:

* виконує валідацію введених значень;
* автоматично замінює кому (,) на крапку (.);
* динамічно підсвічує правильні (styles.inputCorrect) та помилкові (styles.inputIncorrect) значення;
* автоматично скидає підсвітку клітинки після редагування користувачем.

🎨 Графічні компоненти

Містять математичну модель та логіку рендерингу лабораторної установки на Canvas.

Основні можливості:

* синхронізація анімації з поточним етапом виконання роботи;
* реалістичне відображення фізичних процесів;
* інтеграція зі станом елементів керування;
* динамічна зміна положення об'єктів (наприклад, водяного стовпчика) відповідно до перебігу експерименту.

⚙️ Модулі валідації (experimentValidator.ts)

Містять незалежну бізнес-логіку перевірки результатів експерименту.

Функціональність:

* приймають масив даних зі стану React-компонентів;
* виконують розрахунки за еталонними фізичними формулами;
* порівнюють результати користувача з розрахованими значеннями;
* враховують допустиму інженерну похибку вимірювань.
